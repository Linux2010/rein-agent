## 技术决策变更

**放弃 React/Ink hooks 原因**：
- 简化版 Ink 无法正确模拟 React hooks
- 需要 React reconciler 才能支持 hooks
- 直接使用 stdin raw mode + 全局状态更简单可靠

**当前实现**：
- stdin raw mode 处理键盘输入
- setInterval 实现 Spinner 动画
- redraw() 函数重新渲染整个界面
- 保留所有原有功能

## 运行方法

```bash
npm start
# 或
node dist/cli.js
```

**操作**：
- 输入文字 → Enter 发送
- `/help` → 查看命令
- Backspace → 删除字符
- Ctrl+C → 退出
